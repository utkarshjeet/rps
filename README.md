# Rock Paper Scissors Game

A simple **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.
The user selects an option and the computer randomly chooses one. The game then determines whether the user **wins, loses, or draws**.

---

## Features

* Interactive UI
* Random computer move generation
* Instant result display
* Simple and beginner-friendly JavaScript logic
* DOM manipulation using `querySelector` and `addEventListener`

---

## How It Works

1. The user clicks one of the buttons:

   * Rock
   * Paper
   * Scissors

2. JavaScript generates a random choice for the computer.

3. The program compares the **user choice** and **computer choice**.

4. The result is displayed on the screen:

   * Win
   * Lose
   * Draw
